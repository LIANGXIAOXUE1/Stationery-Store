## 一、项目概述

本项目为**校园文具商城平台**，面向学生、办公人群、文具商家及平台管理员，提供线上文具选购、商家入驻、商品管理、订单处理、平台审核与运营管理等一站式服务。

### 目标用户

1.  普通用户：有文具购买需求的学生、办公人群
2.  入驻商家：具备文具经营资质的商户，可进行店铺管理、商品发布、订单发货、数据查看
3.  平台管理员：商城运营与审核管理者，负责商家资质审核、商品上架审核、平台专题 / Banner 运营、用户与数据管理、权限管控

### 核心功能

1.  商家商品管理：发布 / 编辑 / 删除商品，支持图片上传与规格自定义，提交后进入待审核状态
2.  商品审核管理：管理员审核商品，可通过 / 驳回并备注理由，同步通知商家
3.  订单全流程管理：用户下单支付、查看物流、确认收货；商家发货、管理订单状态
4.  管理员综合管理：审核商家资质、管理平台活动、权限控制
5.  文件上传存储：头像、商品图片、商家资质文件管理
